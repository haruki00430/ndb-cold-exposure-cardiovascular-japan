## 2. Climate Data — Japan Meteorological Agency (JMA) / 気象庁月別気象データ

| Item | Detail |
|------|--------|
| **Reference period** | January 2016 – December 2020 (60 months) / 2016年1月〜2020年12月 |
| **Variables** | Monthly mean temperature, precipitation, sunshine hours, cloud cover, humidity |
| **Coverage** | 47 prefectural representative stations / 47都道府県代表地点（札幌〜那覇） |
| **Portal URL** | https://www.data.jma.go.jp/risk/obsdl/index.php |
| **Format** | CSV (Shift-JIS, wide format) |
| **Local path** | `data/raw/weather/jma_monthly_2016_2020.csv` |

**Download steps / ダウンロード手順:**

1. Access the JMA Customized Tables portal above.
2. Select stations: all 47 prefectural representative stations.
3. Select variables: 気温（平均）・降水量・日照時間・雲量・湿度.
4. Set period: January 2016 – December 2020.
5. Download CSV and save to `data/raw/weather/`.

> **Note:** The reference period (2016–2020) is used for structural classification of regional cold exposure patterns. This is independent of the NDB outcome period (FY2023).
